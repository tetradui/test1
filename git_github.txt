Git - это распределенная система контроля версий. 
Предназначен для отслеживания и ведения истории изменения файлов, в вашем проекте.

GitHub - это сайт куда мы можем отправлять все что находится на нашем ноутбуке, чтобы другие люди могли посмотреть на мой проект
и стянуть(скачать) к себе

1) .git innit - чтобы создать скрытую директорию (.git), инициализация
2) git add . - просматривает изменения в нашем проекте, добавляет в промежуточное пространство (index)
3) git commit -m 'massage' -- фиксирует изменения из индекса в локальный репозиторий 
4) git remote add {название ссылки (origin)} - связывает локальный и удаленный репозитории 
5) git push {название связки(origin) main либо master} - отправляет данные из локального в удаленный репозиторий 
6) git pull origin master либо main - отправляет данные из удаленного в локальный репозиторий


Закидываем изменения в готовый репозиторий 
1) git add git_github.txt(название файла)
2) git commit -m 'update file - git_github.txt'
3) git push origin master или main 

Стягиваем изменения из удаленного репозитория 
1) git clone git@github.com:tetradui/test1.git
2) git pull origin master
